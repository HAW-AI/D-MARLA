StateAction  newState 
) throws TechnicalException

Advances a running episode by one step.

Precondition:
An episode must be started by calling startEpisode(). There must be a currently running episode. This means startEpisode() must have been called without a corresponding endEpisode().
Parameters:
rewardForLastStepThe reward the agent has earned for the last action he made.
newStateThe current newState of the agent.
Returns:
Returns an action that is selected out of the possible actions. The selection of the appropriate action is done on the basis of the strategy the agent pursues.
