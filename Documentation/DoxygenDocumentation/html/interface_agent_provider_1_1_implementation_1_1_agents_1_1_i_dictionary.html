Map<StateAction, Float> AgentProvider.Implementation.Agents.IDictionary.getAllSimilarStatesFor ( StateAction  key) throws TechnicalException

Returns a map of all known keys that start with the given key.

For example, if the key is a state, that is compressed as 'AA', all saved StateActions starting with 'AA', like 'AABG', 'AAAA' and so forth will be part of the result.

Note: the given key will be also part of the result, if a value for it was found.

Parameters:
keythe key to that the returned result is supposed to be similar
Returns:
may be empty
Exceptions:
TechnicalExceptionif learning data could not be accessed because of technical difficulties
