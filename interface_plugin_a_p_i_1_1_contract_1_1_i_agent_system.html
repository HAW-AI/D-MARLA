Detailed Description

This interface controls an AgentSystem. It provides methods to start and stop a game, and to get the next GameState from the AgentSystem.

IActionDescription PluginAPI.Contract.IAgentSystem.getActionsForEnvironmentStatus ( IEnvironmentState  current) throws TechnicalException

The agent system will choose it's actions based on the provided current state of the environment. Its decisions will be returned as a new TGameState. This new game state represents the new situation, after the agent has carried out all his actions. startGame must have been called before current != null

Parameters:
